# 환율 API 서버

### Prerequisite
1. Install `mongodb`
2. Install `node`, `npm`
### Settings
1. create `.env` file
    - `.env` should contain: 
    ```
    PORT= ( Port number to run server )
    DB_URI= ( MongoDB URI to run with )
    ```
### How to run
1. run `npm install` to install packages needed
2. run `npm start`
- if error, run `node main.js`
